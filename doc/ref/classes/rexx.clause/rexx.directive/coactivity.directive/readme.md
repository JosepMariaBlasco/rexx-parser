The Coactivity.Directive class (Executor)
==========================================

--------------------------

The Coactivity.Directive class creates objects which
represent the `::COACTIVITY` (or `::CO`) directives,
an Executor-specific directive.

Superclass: [Rexx.Directive](../)

Methods of the Coactivity.Directive class
-----------------------------------------

### directive

![directive](Coactivity.Directive.directive.svg) \

Gets or sets the directive keyword element (`COACTIVITY` or `CO`).
